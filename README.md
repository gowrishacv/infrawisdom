# InfraWisdom - Portfolio

Personal portfolio site for **Gowrisha C. Vishwa Kumar**, Cloud Solutions Architect.

## Stack

- Static HTML/CSS/JS (zero dependencies, zero build step)
- Hosted on GitHub Pages with custom domain `www.infrawisdom.com`
- Apple-inspired full-bleed portfolio layout using existing local assets
- SVG certification logos for Microsoft, HashiCorp Terraform, Google Cloud, and GitHub

## Files

| File | Purpose |
|------|---------|
| `index.html` | Full site: hero, work, impact, projects, architecture, resume, certifications, contact |
| `profile.png` | Profile photo |
| `About.jpg` | Work/portfolio supporting image |
| `Gowrisha Resume.pdf` | Downloadable updated CV |
| `assets/logos/` | Certification logo assets |
| `CNAME` | Custom domain config for GitHub Pages |

## Deploy

Push to `main` branch and GitHub Pages auto-deploys from root.
